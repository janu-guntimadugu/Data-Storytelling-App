Netflix Data Storytelling App



Project Overview



This project analyzes Netflix Movies and TV Shows data and presents meaningful insights through visualizations and storytelling.



Features



\- Dataset Introduction

\- Exploratory Data Analysis (EDA)

\- Pie Chart

\- Line Chart

\- Bar Chart

\- Insights \& Findings

\- Conclusion \& Recommendations



Technologies Used



\- Python

\- Pandas

\- Plotly

\- Streamlit



Dataset



Netflix Titles Dataset



How to Run



1\. Install dependencies:



pip install streamlit pandas plotly



2\. Run the application:



streamlit run app.py
